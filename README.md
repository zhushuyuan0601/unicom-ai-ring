# 联通AI彩铃升级项目

> 🎵 会记录的智能彩铃 - 联通独有 AI 速记联动功能

## 项目简介

本项目是中国联通 AI 彩铃升级的原型 Demo，核心差异化功能是 **AI 速记 + 彩铃联动**，提供"通话前-通话中-通话后"全流程智能服务。

## 核心功能

### 🎬 视频彩铃创作
- **AI 剧本生成**：根据主题自动生成专属剧本
- **智能分镜**：自动拆分镜头，每个镜头独立生成
- **主题模板**：新年祝福、商务形象、生日快乐、节日问候、个性创意、企业形象
- **实时预览**：每个主题有独特的视觉效果和动画

### 🎵 音乐彩铃创作
- **AI 作词**：根据风格自动生成歌词
- **风格选择**：流行、古风、电子、摇滚、爵士、古典
- **波形动画**：音乐生成过程可视化
- **试听界面**：动态波形 + 粒子效果

### 📞 AI 接线员
- **智能应答**：自动接听来电，智能对话
- **留言管理**：来电留言自动转写存储
- **通话记录**：完整的通话历史和摘要

### 📝 AI 速记联动 🌟
- **实时转写**：通话中实时文字转写
- **智能摘要**：通话结束自动生成摘要
- **待办提取**：自动识别待办事项和截止日期
- **关键信息**：自动提取日期、地点、金额

- 科大讯飞 ASR/TTS
